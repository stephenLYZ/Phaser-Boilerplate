## phaser boilerplate

![](http://phaser.io/images/img.png)

> My personal [phaser](http://phaser.io/) boilerplate. 

### Usage

```
$ git clone https://github.com/stephenLYao/Phaser-Boilerplate.git <FoldName>
$ cd <FoldName>
$ (c)npm i
```

development: 

`npm run dev`  

production:  

`npm run build`
